# Weather App Using OpenWeatherMap

**Features**

- Node.js + TypeScript + Express backend
- TypeScript + React frontend
- Uses React-router for routes like /login, /register and /weather
- Uses PostgreSQL to store the database of users
- Uses JWT for authentication
- Concurrently - used to run the client and server at the same time
- Caches weather API results with Redis
- Rate limiting
- Displays weather in a city, has custom units.
